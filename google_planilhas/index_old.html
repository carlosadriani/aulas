Conteúdos para Aulas - Prof. MSc. Carlos Schaeffer

Nível I - Fundamentos e Operações Essenciais

1. Interface e Ciclo de Vida do Dado

Entendimento de linhas, colunas, células, intervalos e tipos de dados (Texto, Número, Data e Booleano). Importação de arquivos .csv e .xlsx.

Exemplo Prático:
Intervalo A1:C10 contendo dados de carga acadêmica. Tipo texto para "Nome da Disciplina" e tipo numérico para "CH Semestral".

2. Fórmulas Básicas e Operadores

Utilização dos operadores aritméticos (+, -, *, /) e funções essenciais de agregação matemática: SOMA, MÉDIA, CONT.NUM e CONT.VALORES.

Fórmula Exemplo:
=MÉDIA(B2:B20)
Calcula o coeficiente de rendimento médio dos alunos no intervalo selecionado.

3. Formatação Condicional

Aplicação de regras visuais automáticas baseadas no conteúdo da célula para construção de dashboards operacionais rápidos.

Exemplo de Regra:
Se o valor da célula for MENOR que 60 (Nota de corte), aplicar preenchimento vermelho claro e texto vermelho escuro.
Nível II - Lógica e Análise de Dados

4. Funções Lógicas (Estruturas de Decisão)

Uso das funções SE, E, OU e SEERRO para criar fluxos condicionais de tratamento de dados e tratamento de exceções em fórmulas.

Fórmula Exemplo:
=SE(B2>=70; "Aprovado"; "Reprovado")
Avalia o status do aluno com base na nota contida na célula B2.

5. Procuras e Correlações (VLOOKUP / XLOOKUP)

Cruzamento de matrizes de dados diferentes utilizando PROCV (VLOOKUP) e o moderno PROCX (XLOOKUP) para busca bidirecional.

Fórmula Exemplo:
=PROCX(A2; 'Alunos'!A:A; 'Alunos'!B:B; "Não Encontrado")
Busca o ID da célula A2 na tabela Alunos e retorna o nome correspondente.

6. Tabelas Dinâmicas e Gráficos

Pivot Tables para sumarização de grandes volumes de dados, segmentação (Filtros) e geração de gráficos de linha, barra e dispersão.

Exemplo Prático:
Agrupar uma planilha de 5.000 logs de segurança por "Tipo de Ataque" nas linhas e "Contagem" nos valores.
Nível III - Recursos Avançados e Engenharia de Dados

7. Manipulação de Matrizes Dinâmicas

Funções poderosas nativas do ecossistema Google: FILTER (filtragem dinâmica), UNIQUE (deduplicação) e SORT (ordenação em tempo de execução).

Fórmula Exemplo:
=FILTER(A2:C100; B2:B100 = "Cloud Computing")
Retorna instantaneamente uma nova sub-tabela apenas com registros da disciplina selecionada.

8. Integração e Consultas Avançadas (QUERY)

Uso da função QUERY que permite executar instruções similares à linguagem SQL (Select, Where, Order By) diretamente em intervalos de dados.

Fórmula Exemplo:
=QUERY(A1:E100; "SELECT A, C WHERE D > 1000 ORDER BY C DESC")
Uma query de banco de dados rodando em tempo real dentro da sua planilha.

9. Automação com Apps Script (Introdução)

Criação de macros e funções personalizadas utilizando JavaScript através do Google Apps Script (GAS) integrado à planilha.

Exemplo de Script:
function onEdit(e) {
  // Registra timestamp automático ao alterar dados
}
