Aula 07 — Extração com FILTER e UNIQUE

FILTER e UNIQUE

Matrizes dinâmicas: tabelas vivas que se atualizam sozinhas

Curso de Planilhas Google — Aula 07

O efeito de "transbordo" (spill)

As funções de matriz são diferentes: você digita a fórmula em uma célula, mas ela cria e preenche várias linhas e colunas automaticamente. Isso é o transbordo.

Atenção ao erro #REF!: como a fórmula precisa de espaço para desenhar a tabela, se houver qualquer texto no caminho dela, ela trava ("o resultado não foi expandido porque colidiria com os dados…"). Basta apagar o texto obstrutor para a tabela aparecer.

Deduplicação com UNIQUE

Numa lista de 2.000 atendimentos, a coluna "Setor" repete "Biblioteca" e "Secretaria" centenas de vezes. A função UNIQUE devolve uma lista limpa, com cada item uma só vez.

=UNIQUE(CadastroGeral!C2:C5000)

A planilha cria instantaneamente a lista de cursos existentes sem repetição. Se um curso novo entrar na lista original, ele aparece aqui automaticamente.

Extração condicional com FILTER

A função FILTER copia para outro local apenas as linhas que atendem a um critério, mantendo o vínculo com a matriz original.

=FILTER(intervalo_dados; condição1; [condição2; ...])

Exemplo — sub-relatório por departamento: na aba Geral, colunas A (Funcionário), B (Cargo), C (Departamento). Em uma aba "RH", célula A1:
=FILTER(Geral!A2:C1000; Geral!C2:C1000 = "Recursos Humanos")
Se o RH contratar alguém na lista geral, a aba do chefe se atualiza sozinha no mesmo segundo.

Combinando FILTER + UNIQUE

Como as funções de matriz conversam entre si, podemos aninhá-las para relatórios mais refinados.

=UNIQUE( FILTER(C2:C1000; B2:B1000 = "Biblioteca") )

Como a planilha lê: primeiro o FILTER isola os tipos de problema apenas do setor "Biblioteca"; depois o UNIQUE remove as repetições — uma lista limpa e cirúrgica para a tomada de decisão do almoxarifado.

Resumindo

As matrizes dinâmicas "transbordam" para preencher tabelas vivas. UNIQUE remove repetições; FILTER extrai linhas por critério, mantendo tudo conectado à origem. Combinadas, geram sub-relatórios automáticos que se atualizam sozinhos — sem copiar e colar.
