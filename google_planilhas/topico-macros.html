O que é uma macro?

Uma macro é uma sequência de ações gravada que você pode repetir com um clique. É como gravar um "passo a passo" e depois mandar a planilha refazê-lo.

Se toda semana você importa um CSV, remove a primeira coluna, formata como moeda, congela o cabeçalho e ordena por nome — uma macro faz tudo isso em segundos, sempre igual, sem você lembrar a sequência.

Toda macro é, na verdade, código

Quando você grava uma macro, o Planilhas Google escreve automaticamente um programa em Apps Script (uma versão do JavaScript). Em Extensões > Macros > Gerenciar macros > Editar você vê algo assim:

function formatarRelatorio() {
  var planilha = SpreadsheetApp.getActiveSheet();
  planilha.getRange('A1:E1').setFontWeight('bold');
  planilha.setFrozenRows(1);
}

Ou seja: gravar macros é uma porta de entrada para a programação. Você pode editar esse código para ajustar a macro sem regravá-la do zero.

Apps Script: indo além da gravação

O Apps Script (Extensões > Apps Script) é o ambiente onde você escreve automações que a gravação de macros não alcança.

Com Apps Script você pode…
Enviar e-mails automáticos a partir dos dados de uma aba
Gerar um PDF do relatório e salvá-lo numa pasta do Drive
Criar menus personalizados na própria planilha
Validar formulários e consolidar várias abas
Rodar tarefas em horários agendados (gatilhos por tempo)

Exemplo: avisar por e-mail quem está com pendência

function avisarPendencias() {
  const aba = SpreadsheetApp.getActiveSheet();
  const dados = aba.getRange('A2:C100').getValues();
  dados.forEach(linha => {
    const [nome, email, situacao] = linha;
    if (situacao === 'Pendente' && email) {
      MailApp.sendEmail(email, 'Documento pendente',
        'Olá ' + nome + ', há um documento pendente.');
    }
  });
}

Esse script percorre a lista, e para cada linha marcada como "Pendente" dispara um e-mail. O que levaria uma tarde de envios manuais roda em segundos.

Aprofundamento: gatilhos (triggers)

Um gatilho faz um script rodar automaticamente, sem ninguém clicar. No editor do Apps Script, ícone de relógio (Acionadores):

Tipo de gatilhoDispara quando…Exemplo de uso
Por tempoEm um horário/intervaloEnviar o resumo do dia toda noite às 18h
Ao abrir (onOpen)Alguém abre a planilhaCriar um menu personalizado
Ao editar (onEdit)Uma célula é alteradaCarimbar a data/hora da última mudança
Ao enviar formulárioChega resposta de um FormsConfirmar inscrição por e-mail

Exemplo de gatilho simples: carimbo de data

function onEdit(e) {
  const linha = e.range.getRow();
  const aba = e.source.getActiveSheet();
  aba.getRange(linha, 6).setValue(new Date());
}

Sempre que alguém edita uma linha, a coluna 6 recebe a data/hora da alteração — um histórico automático de "última atualização".

Funções chamadas onEdit e onOpen são gatilhos simples: rodam sozinhas, sem precisar configurar nada no painel de acionadores.

Permissões, segurança e limites

  • Autorização: na primeira execução, o Google pede que você autorize o script a acessar a planilha, o e-mail etc. Leia o que está sendo pedido antes de aceitar.
  • Cuidado com scripts de terceiros: não cole código de origem desconhecida — ele roda com as suas permissões.
  • Cotas diárias: há limites (ex.: número de e-mails por dia em contas gratuitas/educacionais). Para envios em massa, verifique as cotas.
  • Versionamento: o editor guarda histórico; mantenha uma cópia do script importante.

Macros e scripts ficam dentro do arquivo. Ao compartilhar a planilha como editor, você também compartilha o código — e quem edita pode alterá-lo.

Quando usar macro/script e quando usar fórmula

Precisa de…Melhor ferramenta
Cálculo que se atualiza ao mudar os dadosFórmula (SOMA, QUERY, ARRAYFORMULA)
Repetir uma sequência de cliques/formataçãoMacro gravada
Enviar e-mail, gerar PDF, agendar tarefaApps Script
Reagir a uma edição ou ao abrir o arquivoGatilho (onEdit/onOpen)

Regra prática: se dá para resolver com fórmula, prefira a fórmula (é mais simples e transparente). Reserve macros e scripts para ações e automações que a fórmula não faz.

Resumindo

Macros gravam sequências de ações para repeti-las com um clique (atenção a referências absolutas x relativas). Por baixo, toda macro é Apps Script — que permite enviar e-mails, gerar PDFs, criar menus e rodar tarefas por gatilhos (tempo, onOpen, onEdit). Use fórmulas para cálculos e reserve macros/scripts para ações e automações, sempre atento a permissões e cotas.
