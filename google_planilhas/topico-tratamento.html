Tratamento de Dados

Limpando e organizando antes de analisar

Uso de Dados em Planilhas Google

Dado bruto x dado tratado

Dados importados quase nunca chegam prontos: vêm com espaços sobrando, maiúsculas inconsistentes, duplicatas e colunas misturadas. Tratar é deixá-los limpos e confiáveis antes de somar, filtrar ou cruzar.

Há um ditado na área de dados: "lixo entra, lixo sai". Um relatório só é tão bom quanto a qualidade dos dados que o alimentam. Por isso o tratamento costuma ser a etapa mais demorada — e a mais importante.

Funções para limpar texto

FunçãoO que fazExemplo
ARRUMAR (TRIM)Remove espaços extras antes, depois e duplicados=ARRUMAR(A2)
LIMPAR (CLEAN)Remove caracteres invisíveis de controle=LIMPAR(A2)
MAIÚSCULA / MINÚSCULAPadroniza a caixa do texto=MAIÚSCULA(C2)
PRI.MAIÚSCULA (PROPER)Primeira letra de cada palavra em maiúscula=PRI.MAIÚSCULA(B2)
SUBSTITUIR (SUBSTITUTE)Troca um trecho por outro=SUBSTITUIR(A2;".";"")
VALOR (VALUE)Converte texto numérico em número=VALOR(A2)

É comum combiná-las: =ARRUMAR(MAIÚSCULA(A2)) tira os espaços e padroniza em maiúsculas de uma vez só. Funções podem ser "aninhadas" — uma dentro da outra.

Exemplo: padronizando uma lista

A coluna A veio do formulário com inconsistências. Em B, padronizamos:

A (bruto)B (=ARRUMAR(PRI.MAIÚSCULA(A2)))
2" ana souza "Ana Souza
3"BRUNO lima"Bruno Lima
4"carla DIAS "Carla Dias

Depois de conferir, copie a coluna B e use Colar especial > Somente valores sobre a coluna A para "fixar" o resultado e apagar a coluna auxiliar.

Dividir texto em colunas

Quando vários dados vêm "grudados" numa célula, separe-os:

  • Menu: selecione a coluna > Dados > Dividir texto em colunas > escolha o separador (vírgula, ponto e vírgula, espaço ou personalizado).
  • Fórmula: =DIVIDIRTEXTO(A2;";") (SPLIT) faz o mesmo de forma dinâmica.
A2 = "Ana;ADS;8,5"

=DIVIDIRTEXTO(A2;";")

→ Ana | ADS | 8,5

O inverso — juntar colunas — usa & ou UNIRTEXTO: =A2&" "&B2 ou =UNIRTEXTO(" ";VERDADEIRO;A2:C2).

Remover duplicatas

MétodoComoQuando usar
Limpeza de dadosDados > Limpeza de dados > Remover duplicadosApagar de vez linhas repetidas
ÚNICO (UNIQUE)=ÚNICO(A2:A100)Gerar uma lista sem repetições, dinâmica
CONT.SE=CONT.SE(A$2:A2;A2)>1Marcar/realçar as duplicatas sem apagar

Antes de remover duplicatas de verdade, limpe os espaços e padronize a caixa — senão "Ana " e "ana" serão tratadas como pessoas diferentes e a duplicata não será detectada.

Validação de dados: evitar a sujeira na origem

Melhor que limpar depois é impedir o erro na digitação. Em Dados > Validação de dados você define regras para uma célula:

  • Lista suspensa (dropdown): só permite valores de uma lista (ex.: "Ativa", "Trancada", "Cancelada") — fim das variações de digitação.
  • Intervalo de números: nota entre 0 e 10, por exemplo.
  • Data válida ou texto que contém certo padrão.
  • Rejeitar a entrada inválida ou apenas mostrar um aviso.

Listas suspensas são o recurso mais poderoso para manter uma planilha colaborativa consistente: dez pessoas diferentes preenchendo, mas sempre com os mesmos valores.

Formatação condicional: enxergar problemas

Em Formatar > Formatação condicional, células se colorem sozinhas conforme regras — ótimo para auditar dados:

RegraEfeito
Célula vaziaPintar de amarelo campos não preenchidos
Texto contém erro / "#"Destacar registros problemáticos
Fórmula personalizada =CONT.SE(A:A;A1)>1Realçar duplicatas em vermelho
Escala de coresMapa de calor de notas, gastos, faltas

A formatação condicional não altera o dado — só o visual. É reversível e ideal para revisar uma base antes de fechar um relatório.

Aprofundamento: tratar a coluna inteira com ARRAYFORMULA

Em vez de arrastar uma fórmula por mil linhas, a FÓRMULAMATRICIAL (ARRAYFORMULA) aplica a operação a um intervalo todo de uma vez:

=ARRAYFORMULA(ARRUMAR(MAIÚSCULA(A2:A1000)))

Uma única fórmula na célula B2 já preenche B2 até B1000. Vantagens: mais leve, sem risco de esquecer de arrastar e atualiza sozinha quando chegam linhas novas. É a base de planilhas que "se mantêm limpas".

Aprofundamento: QUERY, o "mini-banco de dados"

A função QUERY usa uma linguagem parecida com SQL para filtrar, ordenar e resumir dados — tratando e analisando em um só passo:

=QUERY(A1:D100; "select B, C where D = 'Ativa' order by C"; 1)

Lê-se: "selecione as colunas B e C, onde a coluna D é 'Ativa', ordenando por C". O último argumento (1) indica que há 1 linha de cabeçalho.

QUERY consegue agrupar e contar: select C, count(B) group by C conta quantos alunos há por curso. É uma das funções mais poderosas do Planilhas Google e dispensa muitas tabelas dinâmicas.

Um fluxo de tratamento recomendado

Importar
Limpar texto
Separar/Tipar
Remover duplicatas
Validar
Analisar

Dica de ouro: mantenha a aba bruta intocada e faça o tratamento em uma aba de trabalho com fórmulas. Assim, se algo der errado, você sempre volta ao dado original — e o processo fica repetível no próximo relatório.

Resumindo

Tratar dados é limpá-los antes de analisar. Use ARRUMAR, MAIÚSCULA, SUBSTITUIR para o texto; Dividir texto em colunas e DIVIDIRTEXTO para separar; remova duplicatas com ÚNICO; previna erros com validação de dados; revise com formatação condicional; e escale com ARRAYFORMULA e QUERY. Dado limpo é relatório confiável.
