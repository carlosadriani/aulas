1. Interface e Introdução aos Dados

Navegação básica, linhas, colunas e a importância de classificar dados de forma correta (nomes de alunos, datas de matrículas e valores de taxas).

Exemplo Prático:
Criação de uma lista de presença e controlo de entrega de documentos para pastas de alunos.

2. Fórmulas de Controle Diário

Uso de operadores matemáticos e funções essenciais como SOMA, MÉDIA, CONT.NUM e CONT.VALORES para estatísticas rápidas.

Fórmula Exemplo:
=SOMA(B2:B20) para totalizar atendimentos realizados na secretaria ao longo do dia.

3. Formatação Condicional

Destacar automaticamente prazos vencidos, faltas críticas ou orçamentos estourados utilizando cores de alerta.

Exemplo Visível:
Pintar de vermelho os alunos com mais de 15 faltas ou processos parados há mais de 5 dias.
Nível II - Lógica Administrativa e Relatórios

4. Funções Lógicas (SE, E, OU)

Automação de respostas baseadas em critérios. Ideal para cálculo automático de resultados acadêmicos ou triagem de requisições.

Fórmula Exemplo:
=SE(C2>=7; "Aprovado"; "Exame") para automatizar o fecho de pautas de notas.

5. Cruzamento de Listas (XLOOKUP / PROCX)

Como encontrar informações de um servidor ou aluno em tabelas diferentes usando o número de identificação ou ID.

Fórmula Exemplo:
Trazer instantaneamente o e-mail institucional do docente a partir de uma lista geral de Recursos Humanos.

6. Tabelas e Gráficos Dinâmicos

Agrupamento de grandes volumes de registos para apresentar relatórios sumarizados.

Exemplo Prático:
Descobrir quantos alunos trancaram o curso por semestre sem precisar contar linha por linha.

7. Extração Automática de Dados com FILTER e UNIQUE

Criação de listas suspensas inteligentes, regras de consistência para entrada de dados e manuseio de filtros para relatórios rápidos.

8. Extração de Relatórios com a Função QUERY

Agrupamento inteligente de dados em larga escala, cruzamento de matrizes acadêmicas e geração de resumos gerenciais automáticos.

9. Introdução à Automação com Apps Script

Automação básica de rotinas com JavaScript para manipulação e envio automatizado de dados contidos nas abas.
