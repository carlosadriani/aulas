Aula 08 — Relatórios com a Função QUERY

Função QUERY

Relatórios escritos como frases de comando

Curso de Planilhas Google — Aula 08

Quando o FILTER não basta

E se você precisar extrair só algumas colunas, em ordem alfabética, trazendo apenas quem atingiu certa pontuação? Juntar várias fórmulas deixaria a planilha lenta.

A função QUERY lê instruções escritas como uma frase estruturada e monta relatórios imediatos. Objetivo: dominar sua escrita e as três cláusulas essenciais — SELECT, WHERE e ORDER BY.

A sintaxe da QUERY

=QUERY(intervalo_dados; "frase_de_comando"; [linhas_de_cabeçalho])
  • intervalo_dados: a tabela completa de onde vêm os dados (ex.: A1:E500).
  • frase_de_comando: o texto entre aspas duplas com as instruções.
  • linhas_de_cabeçalho: quantas linhas de título há no topo (geralmente 1).

As três cláusulas principais

CláusulaO que fazTradução prática
SELECTIndica quais colunas trazer"Traga só as colunas A e C, pule o resto"
WHEREAplica as regras de filtragem"Só mostre quem é da Secretaria com prazo > 30 dias"
ORDER BYOrganiza a classificação"Ordene do maior valor para o menor"

As palavras-chave (SELECT, WHERE, ORDER BY) ficam sempre em maiúsculas, dentro das aspas.

Cenário: painel de auditoria de docentes

Planilha "CadastroDocentes": A (Nome), B (Faculdade/Centro), C (Regime), D (Turmas). O diretor quer Nome e Regime, só dos professores da faculdade de "Tecnologia", em ordem alfabética:

=QUERY(CadastroDocentes!A1:D500; "SELECT A, C WHERE B = 'Tecnologia' ORDER BY A ASC"; 1)
TrechoO que faz
SELECT A, COculta as colunas B e D no resultado
WHERE B = 'Tecnologia'Filtra a tabela (texto entre aspas simples)
ORDER BY A ASCOrdena os nomes de A a Z

Cuidados especiais e regras de ouro

  • Colunas em maiúsculas: use SELECT A, nunca SELECT a.
  • Aspas simples para texto: como a frase já está entre aspas duplas, o texto comparado usa aspas simples — WHERE C = 'Integral'.
  • Números sem aspas: WHERE D > 3 (mais de 3 turmas).

Exemplo combinado (texto + número) — professores em regime 'Integral' com mais de 4 turmas:
=QUERY(A1:D500; "SELECT A WHERE C = 'Integral' AND D > 4"; 1)

Resumindo

A QUERY monta relatórios com uma frase de comando: SELECT escolhe colunas, WHERE filtra e ORDER BY ordena. Lembre das regras: colunas em maiúsculas, texto entre aspas simples e números sem aspas. É quase um mini-banco de dados dentro da planilha.
