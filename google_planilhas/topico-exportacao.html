Levando informações da planilha para o mundo

Uso de Dados em Planilhas Google

Por que exportar?

Nem sempre quem recebe seus dados usa o Planilhas Google. Exportar é converter sua planilha em um formato que outra pessoa, sistema ou impressora consiga ler.

Você exporta para entregar um relatório em PDF à diretoria, para enviar um CSV a um sistema que vai importá-lo, para abrir no Excel de um colega ou para publicar uma tabela num site. Cada finalidade pede um formato diferente.

Arquivo > Fazer download

O caminho principal é o menu Arquivo > Fazer download, que oferece:

FormatoExtensãoUse quando…
Microsoft Excel.xlsxO destinatário usa Excel e precisa de fórmulas/abas
OpenDocument.odsO destino usa LibreOffice/software livre
PDF.pdfÉ um relatório para ler/imprimir, sem edição
Página da web.htmlVai publicar a tabela num site
Valores separados por vírgula.csvOutro sistema vai importar os dados (só a aba atual)
Valores separados por tabulação.tsvSistema que espera tabulação como separador

CSV e TSV exportam apenas a aba ativa e somente os valores — fórmulas viram resultados, cores e gráficos se perdem. Para preservar tudo, use XLSX ou ODS.

O que cada formato preserva

RecursoXLSX/ODSPDFCSV/TSV

FórmulasSimNão (só resultado)Não (só resultado)

Regra prática: PDF para ler, XLSX para editar, CSV para alimentar outro sistema.

Exportar um PDF bem feito

Ao escolher Arquivo > Fazer download > PDF, abre-se uma tela de ajustes que define a qualidade do relatório:

  • Exportar: planilha inteira, apenas a aba atual ou apenas as células selecionadas.
  • Orientação: retrato ou paisagem (tabelas largas pedem paisagem).
  • Tamanho do papel e margens: A4 é o padrão; margens estreitas cabem mais.
  • Ajustar à largura: evita que colunas "vazem" para outra página.
  • Quebras de página, linhas de grade, cabeçalhos/rodapés com número de página e data.
  • Repetir linhas de cabeçalho congeladas em todas as páginas.

Congelar a linha de títulos (Visualizar > Congelar > 1 linha) faz o cabeçalho se repetir em cada página do PDF — essencial em listas longas de alunos ou servidores.

Exportar só uma parte dos dados

Nem sempre você quer o arquivo inteiro. Para exportar um recorte:

  • Selecione o intervalo desejado antes de exportar e escolha "células selecionadas" (no PDF).
  • Esconda colunas/abas sigilosas antes de gerar o arquivo.
  • Crie uma aba-resumo só com o que será compartilhado e exporte apenas ela em CSV.

Ao enviar dados para fora, lembre-se da LGPD: remova CPF, telefones e outros dados pessoais que não sejam necessários para quem vai receber o arquivo.

Compartilhar não é a mesma coisa que exportar

ControleLeitor / Comentarista / EditorQuem tem o arquivo faz o que quiser

No botão Compartilhar, você define o papel de cada pessoa (Leitor, Comentarista, Editor) e pode restringir o download/cópia para leitores — um meio-termo entre os dois mundos.

Publicar na web

Arquivo
Compartilhar
Publicar na web

A opção Publicar na web gera um link público ou um código <iframe> para embutir a tabela/gráfico em um site. Você escolhe publicar a planilha toda ou só uma aba, como página web ou como CSV/XLSX.

O conteúdo publicado se atualiza automaticamente (com pequeno atraso). Um gráfico publicado pode ser incorporado num portal e refletirá sempre os dados mais recentes da planilha.

Publicar torna o conteúdo acessível a qualquer pessoa com o link, sem login. Nunca publique abas com dados pessoais ou sigilosos.

Aprofundamento: exportação automática por URL

Toda planilha tem URLs especiais que geram um arquivo exportado na hora — úteis para integrações e para links de "baixar relatório":

.../export?format=pdf → PDF da planilha
.../export?format=xlsx → pasta Excel
.../export?format=csv&gid=0 → CSV da aba de gid 0

Com Apps Script (visto no tópico de Macros), dá para programar a exportação: gerar um PDF do relatório do mês e enviá-lo por e-mail automaticamente, sem ninguém clicar em nada.

Resumindo

Exportar converte a planilha em um arquivo para outro uso. Escolha o formato pela finalidade: PDF para ler/imprimir, XLSX/ODS para editar preservando tudo, CSV/TSV para alimentar sistemas. Compartilhar mantém o dado vivo; Publicar na web divulga uma tabela pública. E lembre sempre da LGPD ao mandar dados para fora.
