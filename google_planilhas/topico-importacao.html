Uso de Dados em Planilhas Google

Por que importar dados?

No dia a dia da universidade, raramente os dados nascem dentro da planilha. Eles chegam de sistemas acadêmicos, do RH, da biblioteca, de formulários e de relatórios em outros formatos.

Importar é o ato de trazer esses dados externos para dentro do Planilhas Google, convertendo-os em linhas e colunas que você pode somar, filtrar e cruzar. Saber importar bem evita o trabalho manual de redigitar centenas de registros — e os erros que vêm junto.

Vamos ver dois grandes caminhos: a importação manual (menu Arquivo > Importar) e a importação por fórmulas (funções IMPORT…), que se atualizam sozinhas.

O CSV é o "idioma universal" das planilhas: qualquer sistema sabe gerá-lo e qualquer programa sabe lê-lo. Por isso é o formato que você mais vai encontrar ao puxar relatórios.

Entendendo um arquivo CSV por dentro

Um CSV é apenas texto. Cada linha do arquivo vira uma linha da planilha, e cada separador marca o fim de uma coluna. Abra-o no Bloco de Notas e verá algo assim:

No Brasil, a vírgula costuma ser o separador decimal (R$ 1.250,50). Por isso muitos sistemas usam ponto e vírgula (;) como separador de colunas. Fique atento a isso na hora de importar.

matricula;nome;curso;media
20231;Ana Souza;ADS;8,5
20232;Bruno Lima;ADS;6,0
20233;Carla Dias;Redes;9,2

Esse mesmo conteúdo, depois de importado, vira uma tabela de 4 colunas e 3 linhas de dados. O Planilhas Google detecta o separador automaticamente na maioria dos casos, mas permite escolhê-lo manualmente.

Importando planilhas do Excel (.xlsx)

Arquivos .xlsx podem ser abertos de duas formas:

  • Abrir como Planilhas Google: no Drive, dê dois cliques no arquivo e use Abrir com > Planilhas Google. Isso cria uma cópia convertida, preservando fórmulas compatíveis, abas e formatação.
  • Importar para dentro de uma planilha existente: use Arquivo > Importar para trazer apenas os dados de uma aba específica.

Nem tudo converte 100%: macros do Excel (VBA), alguns gráficos e funções muito específicas podem não funcionar. Fórmulas comuns (SOMA, SE, PROCV) são convertidas para os equivalentes em português (SOMA, SE, PROCV/PROCX) automaticamente.

Copiar e colar de outras fontes

Muitas vezes os dados estão num e-mail, num PDF ou numa página web. Você pode simplesmente copiar (Ctrl+C) e colar (Ctrl+V) na planilha. Para controlar o resultado, use Colar especial:

Colar especialO que cola
Somente valores (Ctrl+Shift+V)Apenas o texto/número, sem formatação nem fórmulas
Somente formatoSó cores, bordas e fontes — útil para padronizar
TranspostoTroca linhas por colunas

Ao colar texto "grudado" numa coluna só, selecione a coluna e use Dados > Dividir texto em colunas para separá-lo. Veremos isso em detalhe no tópico de Tratamento de Dados.

Importação viva: as funções IMPORT…

Diferente da importação manual, as funções IMPORT… criam uma ligação dinâmica: quando a fonte muda, a planilha se atualiza sozinha.

FunçãoPara que serve
IMPORTDATAImporta um arquivo CSV ou TSV publicado em uma URL
IMPORTHTMLImporta uma tabela ou lista de uma página web
IMPORTXMLImporta dados específicos via caminho XPath
IMPORTFEEDImporta um feed RSS/Atom (notícias, blogs)
IMPORTRANGEImporta intervalos de outra planilha do Google (tópico próprio)

IMPORTDATA: puxar um CSV da web

=IMPORTDATA("https://dados.exemplo.gov.br/matriculas.csv")

A função busca o arquivo na URL e despeja todo o conteúdo a partir da célula onde foi digitada. É ideal para dados abertos de órgãos públicos ou relatórios que o sistema publica num link fixo.

Você pode indicar o separador e a localidade:
=IMPORTDATA("https://...arquivo.csv"; ";"; "pt_BR")
Assim o ponto e vírgula é tratado como separador de colunas e a vírgula como decimal.

IMPORTHTML: capturar uma tabela de um site

=IMPORTHTML("https://pt.wikipedia.org/wiki/Brasil"; "table"; 1)

Os três argumentos são: a URL, se você quer uma "table" ou uma "list", e o número de ordem do elemento na página (a 1ª tabela, a 2ª, etc.).

Excelente para trazer cotações, rankings, calendários acadêmicos publicados em HTML ou listas de feriados sem copiar e colar manualmente — e o dado se atualiza quando a página muda.

Se a tabela vier vazia, tente outro índice (2, 3…): a "tabela 1" da página nem sempre é a que você vê. E a função só funciona com páginas públicas que não exigem login.

IMPORTXML: extrair um dado específico

=IMPORTXML("https://exemplo.com/cotacao"; "//span[@id='dolar']")

Quando você não quer a tabela inteira, mas só um pedaço da página (um preço, um título, uma data), o IMPORTXML usa um caminho XPath para apontar exatamente onde está o dado.

XPath é como um "endereço" dentro do código da página. Exemplos comuns: //h1 (todos os títulos), //a/@href (todos os links), //div[@class='preco'] (divs com a classe "preco"). É a função mais poderosa e a mais técnica do grupo.

Exemplo prático: relatório de matrículas

A Secretaria recebe do sistema acadêmico um arquivo matriculas_2026-1.csv. Depois de Arquivo > Importar > Fazer upload, a aba fica assim:

ABCD
1matriculanomecursosituacao
220231Ana SouzaADSAtiva
320232Bruno LimaADSTrancada
420233Carla DiasRedesAtiva

A partir daqui, em segundos é possível contar matrículas ativas com =CONT.SE(D2:D4;"Ativa"), sem conferir registro por registro. Importar bem é o primeiro passo de todo relatório.

Resumindo

Importar é trazer dados externos para a planilha. Use Arquivo > Importar para arquivos (CSV, XLSX, ODS), prestando atenção ao separador, à codificação e ao local de importação. Para ligações que se atualizam sozinhas, use as funções IMPORTDATA, IMPORTHTML e IMPORTXML. Com os dados dentro da planilha, o trabalho de análise pode começar.
