Cloud Computing

Aula 01: Fundamentos, História e Glossário

Curso: Análise e Desenvolvimento de Sistemas

O que é Cloud Computing?

É o fornecimento de serviços de computação (servidores, armazenamento, bancos de dados, rede, software) pela Internet ("a nuvem").


Modelo de Utilidade: Assim como a energia elétrica, você consome o que precisa e paga apenas pelo que foi utilizado.

Evolução Histórica

1960: Mainframes

Surgimento do Time-Sharing: múltiplos usuários dividindo o poder de processamento de uma única máquina central.

1999: Salesforce

Primeiro grande exemplo de Software como Serviço (SaaS) entregue via navegador.

2006: AWS

Lançamento do EC2 e S3. A infraestrutura passa a ser tratada como código e serviço.

Hoje: Cloud Native

Uso de Containers, Microserviços e Inteligência Artificial integrada na infraestrutura.

Glossário Essencial

TermoDefinição
NISTÓrgão que define os padrões de Cloud (5 características, 3 modelos).
CapExInvestimento em ativos físicos (comprar o servidor).
OpExDespesa operacional (pagar a conta mensal da nuvem).
ElasticidadeCapacidade de aumentar/diminuir recursos automaticamente.
MultitenancyVários clientes compartilhando o mesmo recurso físico isoladamente.

Verificação de Aprendizagem

Se uma aplicação aumenta o número de servidores durante a Black Friday, qual característica NIST está sendo usada?

A) Amplo Acesso à Rede
B) Pool de Recursos
C) Elasticidade Rápida
D) Autoatendimento
Dica: A resposta correta é a letra C.
