19. Sockets e Jogos em Rede

Abstração de baixo nível para comunicação entre processos na rede. Estudo de fluxos stream síncronos e assíncronos aplicados a jogos multiplayer em tempo real.

01. O que são Sockets de Rede?

Um Socket é uma abstração do sistema operacional que funciona como um ponto de extremidade (endpoint) para comunicação bidirecional entre processos rodando em máquinas diferentes. Ele serve como interface entre a Camada de Aplicação (7) e a Camada de Transporte (4).

Arquitetura de Endereçamento: Um socket de internet é identificado de forma única pela combinação de um **Endereço IP** (camada de rede) e um **Número de Porta** (camada de transporte). Ex: 127.0.0.1:5000.

02. Ciclo de Vida das Conexões Stream (TCP)

Para estabelecer um canal confiável orientado a fluxo de bytes (TCP), o servidor e o cliente executam primitivas específicas do sistema operacional:

  • Servidor: Executa socket() para instanciar, bind() para amarrar à porta local, listen() para entrar em estado de escuta passiva, e accept() que bloqueia a execução até a chegada de um handshake de cliente.
  • Cliente: Executa connect() disparando ativamente o Handshake de 3 Vias do TCP para abrir o canal estável.

03. Servidor Multi-Cliente e Cliente CLI em Python

Abaixo encontra-se a implementação completa de um **Servidor Multithreading TCP** em Python. Ele gerencia as regras de negócio de um jogo da velha, coordena os estados de dois jogadores e valida as vitórias ou empates.

Código do Servidor (`server.py`)
import socket
import threading

HOST = '127.0.0.1'
PORT = 5000

# Estado global do Jogo da Velha
jogadores = []  # Lista com as conexões dos 2 clientes
tabuleiro = [" "] * 9
turno_atual = 0  # 0 para Jogador X, 1 para Jogador O

def verificar_vencedor():
    linhas_vitoria = [
        [0, 1, 2], [3, 4, 5], [6, 7, 8],  # Linhas
        [0, 3, 6], [1, 4, 7], [2, 5, 8],  # Colunas
        [0, 4, 8], [2, 4, 6]              # Diagonais
    ]
    for linha in lines_vitoria:
        if tabuleiro[linha[0]] == tabuleiro[linha[1]] == tabuleiro[linha[2]] != " ":
            return tabuleiro[linha[0]]
    if " " not in tabuleiro:
        return "Empate"
    return None

def transmitir(mensagem):
    """Envia uma mensagem de texto para ambos os jogadores"""
    for j in jogadores:
        try:
            j.sendall((mensagem + "\n").encode('utf-8'))
        except:
            pass

def gerenciar_partida():
    global turno_atual
    transmitir("JOGO_INICIADO")
    jogadores[0].sendall("VOCE_E_X\nSUA_VEZ\n".encode('utf-8'))
    jogadores[1].sendall("VOCE_E_O\nAGUARDE_ADVERSARIO\n".encode('utf-8'))

    while True:
        conn_atual = jogadores[turno_atual]
        conn_adversario = jogadores[1 - turno_atual]
        simbolo = "X" if turno_atual == 0 else "O"

        try:
            data = conn_atual.recv(1024).decode('utf-8').strip()
            if not data:
                break
            
            # Processa o movimento enviado pelo cliente (ex: "4")
            posicao = int(data)
            if tabuleiro[posicao] == " ":
                tabuleiro[posicao] = simbolo
                
                status = verificar_vencedor()
                if status:
                    transmitir(f"MOVIMENTO:{posicao}:{simbolo}")
                    if status == "Empate":
                        transmitir("FIM:EMPATE")
                    else:
                        transmitir(f"FIM:VENCEDOR:{status}")
                    break
                else:
                    # Informa o lance para continuar o jogo
                    conn_adversario.sendall(f"MOVIMENTO:{posicao}:{simbolo}\nSUA_VEZ\n".encode('utf-8'))
                    conn_atual.sendall(f"AGUARDE_ADVERSARIO\n".encode('utf-8'))
                    turno_atual = 1 - turno_atual
            else:
                conn_atual.sendall("POSICAO_INVALIDA\nSUA_VEZ\n".encode('utf-8'))
        except Exception as e:
            print(f"Erro na partida: {e}")
            break

def iniciar_servidor():
    server = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    server.bind((HOST, PORT))
    server.listen(2)
    print(f"[*] Servidor de Tic-Tac-Toe rodando em {HOST}:{PORT}")

    while len(jogadores) < 2:
        conn, addr = server.accept()
        jogadores.append(conn)
        id_jogador = len(jogadores)
        print(f"[+] Jogador {id_jogador} conectado de {addr}")
        
        if id_jogador == 1:
            conn.sendall("AGUARDAR_INICIO\n".encode('utf-8'))
        elif id_jogador == 2:
            threading.Thread(target=gerenciar_partida).start()

if __name__ == "__main__":
    iniciar_servidor()
Código do Cliente CLI em Python (`client.py`)

Um exemplo funcional de cliente texto puro em Python para teste imediato das chamadas:

import socket
import threading
import sys

def desenhar_tabuleiro(tab):
    print(f"\n {tab[0]} | {tab[1]} | {tab[2]} ")
    print("---|---|---")
    print(f" {tab[3]} | {tab[4]} | {tab[5]} ")
    print("---|---|---")
    print(f" {tab[6]} | {tab[7]} | {tab[8]} \n")

def escutar_servidor(client_socket):
    tabuleiro = [" "] * 9
    try:
        while True:
            linhas = client_socket.recv(1024).decode('utf-8').split('\n')
            for linha in linhas:
                if not linha: continue
                
                if "AGUARDAR_INICIO" in linha:
                    print("[Server] Aguardando o segundo jogador se conectar...")
                elif "JOGO_INICIADO" in linha:
                    print("[Server] Partida iniciada!")
                elif "VOCE_E_" in linha:
                    print(f"[Info] {linha}")
                elif "MOVIMENTO" in linha:
                    _, pos, simb = linha.split(':')
                    tabuleiro[int(pos)] = simb
                    desenhar_tabuleiro(tabuleiro)
                elif "SUA_VEZ" in linha:
                    desenhar_tabuleiro(tabuleiro)
                    posicao = input("Sua vez! Escolha uma posição (0-8): ")
                    client_socket.sendall((posicao + "\n").encode('utf-8'))
                elif "AGUARDE_ADVERSARIO" in linha:
                    print("[Info] Aguardando a jogada do oponente...")
                elif "FIM" in linha:
                    print(f"\n[FIM DE JOGO] {linha}")
                    return
    except Exception as e:
        print(f"Conexão perdida: {e}")
    finally:
        client_socket.close()

def iniciar_cliente():
    client = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    client.connect(('127.0.0.1', 5000))
    print("[*] Conectado ao servidor!")
    threading.Thread(target=escutar_servidor, args=(client,), daemon=True).start()
    
    # Mantém o processo principal vivo
    try:
        while True: pass
    except KeyboardInterrupt:
        client.close()

if __name__ == "__main__":
    iniciar_cliente()

04. Cliente Assíncrono em Dart (Console/Flutter Engine)

Abaixo está o código em **Dart** focado em processamento reativo de streams, compatível com aplicações de console e facilmente integrável ao ciclo de estados do Flutter.

import 'dart:io';
import 'dart:convert';

void desenharTabuleiro(List<String> tab) {
  print("\n ${tab[0]} | ${tab[1]} | ${tab[2]} ");
  print("---|---|---");
  print(f" ${tab[3]} | ${tab[4]} | ${tab[5]} ");
  print("---|---|---");
  print(f" ${tab[6]} | ${tab[7]} | {tab[8]} \n");
}

void main() async {
  final String host = '127.0.0.1';
  final int porta = 5000;
  List<String> tabuleiro = List.filled(9, " ");

  print("[*] Renderizando Grid Inicial...");
  desenharTabuleiro(tabuleiro);

  try {
    print("[*] Solicitando conexão ao servidor TCP ($host:$porta)...");
    final socket = await Socket.connect(host, porta);
    print("[+] Conectado com sucesso!");

    // Ouve as mensagens emitidas pelo servidor de forma assíncrona
    socket.transform(utf8.decoder).transform(const LineSplitter()).listen(
      (String linha) {
        if (linha.isEmpty) return;

        // Regra: Quando receber mensagem do servidor, mostra ao usuário
        if (linha.contains("AGUARDAR_INICIO")) {
          print("[Mensagem Servidor] Aguarde o início do jogo. Aguardando oponente...");
        } else if (linha.contains("JOGO_INICIADO")) {
          print("[Mensagem Servidor] O jogo começou!");
        } else if (linha.contains("VOCE_E_")) {
          print("[Regra de Jogo] $linha");
        } else if (linha.contains("AGUARDE_ADVERSARIO")) {
          print("[Status] Aguarde o lance do adversário...");
        } else if (linha.contains("MOVIMENTO")) {
          var partes = linha.split(':');
          int pos = int.parse(partes[1]);
          String simb = partes[2];
          tabuleiro[pos] = simb;
          desenharTabuleiro(tabuleiro);
        } else if (linha.contains("SUA_VEZ")) {
          desenharTabuleiro(tabuleiro);
          stdout.write("Sua vez de jogar! Digite a posição desejada (0-8): ");
          String? entrada = stdin.readLineSync();
          if (entrada != null && entrada.isNotEmpty) {
            socket.write("$entrada\n");
          }
        } else if (linha.contains("POSICAO_INVALIDA")) {
          print("[Erro] Posição já preenchida! Tente outra.");
        } else if (linha.contains("FIM")) {
          print("\n[PARTIDA ENCERRADA] $linha");
          socket.destroy();
          exit(0);
        }
      },
      onError: (err) {
        print("[Erro] Falha no fluxo: $err");
        exit(1);
      },
      onDone: () {
        print("[-] Conexão encerrada pelo servidor remoto.");
        exit(0);
      },
    );

  } catch (e) {
    print("[Erro Crítico] Não foi possível conectar ao servidor: $e");
  }
}

05. Análise de Protocolo: Mensagens de Controle

Na engenharia de sistemas de redes, estabelecer um **Protocolo de Aplicação customizado** sobre o fluxo bruto do TCP é obrigatório. Veja a especificação das strings de controle adotadas neste laboratório:

Comando / Payload Origem → Destino Ação na Infraestrutura
AGUARDAR_INICIO Servidor → Cliente 1 Informa ao primeiro cliente que a conexão TCP foi aceita, mas o jogo precisa de um oponente.
VOCE_E_X / VOCE_E_O Servidor → Clientes Define a atribuição de papéis e símbolos para o tratamento lógico dos lances.
MOVIMENTO:posicao:simbolo Servidor → Adversário Atualiza o grid de exibição local do oponente sem necessidade de trafegar o tabuleiro inteiro.
FIM:VENCEDOR:X Servidor → Ambos Encerra a escuta do Socket em virtude do atingimento de um estado terminal de jogo.
