Laboratório Prático — Exercícios de bases numéricas (Aula 01)

1. Binário ➔ Decimal

  1. Converta o binário 1010
  2. Converta o binário 11001
  3. Converta o binário 101010
  4. Converta o binário 11111111 Dica: este é o valor máximo de 1 Byte (8 bits).

2. Decimal ➔ Binário

Utilize o método das divisões sucessivas:

  1. Converta o decimal 13
  2. Converta o decimal 25
  3. Converta o decimal 64
  4. Converta o decimal 128

3. O Mundo Hexadecimal

  1. Converta o binário 1010 1111 diretamente para Hexadecimal. Dica: Divida em dois grupos de 4 bits (nibbles).
  2. Qual o valor decimal do hexadecimal 1A?
  3. Cores Web: Se a cor branca é #FFFFFF, qual o valor decimal de cada canal (R, G, B) individualmente?

4. Desafio

  1. Um arquivo de configuração possui 2 KB. Quantos Bytes ele possui exatamente? E quantos Bits?
  2. Se um processador antigo possui um barramento de endereços de apenas 4 bits, quantas posições de memória (endereços) diferentes ele consegue acessar no máximo?
